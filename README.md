# Sistema de Gestion de Eventos

## Integrantes
- Luis Antonio Ruiz Duran RD20014
- Edwin Eduardo Torres Perez TP21002

## Descripcion
Sistema desarrollado en PSeInt que permite crear eventos, registrar asistentes y controlar la capacidad.

## Funcionalidades
- Crear eventos
- Registrar asistentes
- Mostrar eventos
- Control automatico de capacidad

## Logica del sistema
Cada evento tiene una capacidad maxima. Cuando se alcanza el limite, el evento se cierra automaticamente.
